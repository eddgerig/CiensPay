# CIENSPAY
Este repositorio contiene el desarrollo del sistema para la entidad bancaria dentro de un ecosistema simulado de pagos electrónicos básicos con integración interbancaria el cual hemos denominado "CiensPay". El objetivo principal es emular el rol de un banco emisor y adquiriente en transacciones de comercio electrónico.

## Docker
Para ejecutar este proyecto con Docker utilice el siguiente comando: 
docker compose up -d

## Versiones

Node: 22-alpine
Django: python:3.11-slim
